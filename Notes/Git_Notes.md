## Clone a Repo
git clone https://github.com/harishbabus/AI-Engineer-Blueprint.git
## Prepare all the changes to push to Git
git add .
## Take snapshot of all the files pushed
git commit -m "Day 01 - Python Basics"
## Uploads local commits from computer to a remote server
git push
## Check file changes
git status
## Upload local commits to the main branch of online repository
git push origin main
